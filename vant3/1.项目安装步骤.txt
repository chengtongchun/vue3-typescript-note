vue3 typescript vant31.建项目 d:盘上建个目录 进入该目录  vue create blibli配置：      (1)选自定义，(不选vue2和vue3)	(2)空格键 选babel,typescript,router,vuex,css,linter,回车键下一步骤	(3)3.x	(4)no,yes,yes,less,standard,lint on save, in dedicated config files 继续回车键2.用vscode打开该目录下面blibli3.整理目录  在src下assets下面建三个目录     styles中放样式表      建images放图片     建font下载iconfont图标  复制老师assets下的文件4.按需引入vant3   1.找vant3的官网http://vant3.uihtm.com/   2.找快速上手    (1)  npm i vant 安装vue3Vant    (2)安装插件： npm i babel-plugin-import -D    (3) babel.config.js 中添加配置{  "plugins": [    [      "import",      {        "libraryName": "vant",        "libraryDirectory": "es",        "style": true      }    ]  ]}     (4)在main.js中引入组件按需引入// 原始代码import { Button } from 'vant';// 编译后代码import Button from 'vant/es/button';import 'vant/es/button/style';导入所有组件（不推荐）Vant 支持一次性导入所有组件，引入所有组件会增加代码包体积，因此不推荐这种做法。import { createApp } from 'vue';import Vant from 'vant';import 'vant/lib/index.css';const app = createApp();app.use(Vant);
